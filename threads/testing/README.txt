Για την εκτέλεση του DEMO εκτελέστε το bash script "run_tests" και στο φάκελο stdout θα εμφανιστούν οι εξόδου από τον server και τους 3 clients τους οποίους ανοίγει με ονόματα giannis, glafkos και spyros. ΠΡΟΣΟΧΗ: το bash script αρχικά διαγράφει κάτω από τον φάκελο /tmp/ αν υπάρχουν αρχεία με ονόματα giannis, spyros και glafkos καθώς θέλει να δημιουργήσει fifo τα οποία θα λειτουργούν σαν το stdin των 3 clients. ΠΑΡΑΤΗΡΗΣΗ: Η υλοποίηση με τα threads λειτουργεί κανονικά απλά έχουν κάποιο πρόβλημα τα script και δεν γίνεται σωστά redirect τα stdout στα αρχεία, με αποτέλεσμα τα αρχεία τα οποία δημιουργούνται κάτω από τον φάκελο stdout/ να είναι κενά.
